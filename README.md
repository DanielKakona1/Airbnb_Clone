# Airbnb_Clone
Airbnb_clone is a clone of the Airbnb system.

## 🚀 Techs

- ⚛️ Reactjs

## ✋🏻 Prerequisites

- [Node.js](https://nodejs.org/)

## 🔥 Install & Execute

1. Clone the repo;
2. Run `cd Airbnb_Clone` to open the cloned repo;
3. Run `yarn install` to install the dependecies;
4. Run `yarn start` to run the system;
